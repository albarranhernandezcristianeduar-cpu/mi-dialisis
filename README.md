# mi-dialisis
App de Control de Diálisis Peritoneal. Herramienta PWA para registro de sesiones médicas. Incluye: captura de métricas (UF, Glucosa, T.A.), alertas de salud, diseño orgánico, guardado permanente sin internet y generación de reportes PDF profesionales para el médico. 🩺✨ #Salud #PWA #JavaScript
